# Asset ID Naming

示例命名：

- `icon.room.mine`
- `icon.room.chest`
- `icon.room.exit`
- `icon.room.event`
- `icon.room.monster`
- `sprite.player.default`
- `tile.room.base_floor`
- `tile.room.base_wall`
- `ui.panel.default`
- `ui.button.default`
- `sfx.mine.trigger`
- `sfx.ui.confirm`
- `music.run.ambient_01`

规则：

- `asset_id` 使用小写。
- `asset_id` 使用点号分层。
- `asset_id` 不使用中文。
- Godot 文件名使用小写蛇形命名。
- 原始中文说明写在 `note` 字段。
